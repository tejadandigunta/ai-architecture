No memory. Just context.

An LLM does not remember your chat like a person. It only sees the text that fits inside its current window.

Every reply gets the whole visible chat

Your new message is sent with the conversation so far: your inputs, the model's outputs, instructions, files, and tool results.

The window is the model's short-term view

It can only attend to a fixed number of tokens at once. Anything outside that window is invisible.

03

The limit exists because attention is expensive

Attention compares tokens with other tokens. More context means a much bigger comparison map and much higher compute cost.

Full
As the visible chat fills up, the model has more tokens to compare before it can answer.
04

Cross the limit, old context falls out

When the conversation gets too long, earlier tokens can be removed or summarized. To the model, those details are gone.

name 2 hrs old new task latest reply files current ask tools
It did not forget like a human. It just cannot see that older text anymore.
05

Bigger windows change what models can do

A larger context window means more code, documents, chat history, and examples can be visible in one pass.

The simple version

The context window is the model's working table. It can reason over whatever is on the table, but once something is pushed off, it cannot use it.
